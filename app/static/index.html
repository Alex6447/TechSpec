// Quick Start

🚀 Быстрый старт

bash
# 1. Настройка окружения
cp .env.example .env
# Заполнить: DATABASE_URL, MASTER_KEY (32 байта), ADMIN_TOKEN, BOT_TOKEN, ARQEN_BASE_URL

# 2. Запуск стека
docker-compose up --build -d

# 3. Применить миграции
docker-compose exec app alembic upgrade head

# 4. Проверка Admin API
curl http://localhost:8000/api/v1/admin/accounts \
  -H "X-Admin-Token: your_admin_token"
# → {"total": 0, "used": 0, "free": 0, "accounts": []}

// Security

🔒 Безопасность

🔐
Encrypted at Rest
RSA private keys и access_token хранятся зашифрованными (AES-256-GCM)
🎲
Уникальный Nonce
Каждый вызов encrypt() генерирует новый 12-байтный nonce — семантически безопасно
⏱
Timing-Safe Compare
Admin token сравнивается через secrets.compare_digest() — защита от timing attacks
📦
JWE Card Details
PAN/CVV/Expiry передаются от Arqen в зашифрованном JWE (RSA-OAEP-256 + A256GCM)
🏷
JWT TTL 5 минут
Assertion JWT живёт 5 минут, jti уникален для каждого запроса
🌍
Sandbox Only
Вся разработка и тестирование ведётся в изолированном Arqen Sandbox
